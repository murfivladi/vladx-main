# VladX - Мощный программный язык с русским синтаксисом

VladX - это интерпретируемый язык программирования с русским синтаксисом, написанный на Node.js. Язык поддерживает современные функции программирования и ориентирован на простоту и читаемость кода.

## Установка

### Автоматическая установка из ZIP архива (рекомендуется)

Новый установщик позволяет загружать VladX из ZIP архива с вашего сайта:

```bash
# Установка из стандартного URL
curl -fsSL https://your-website.com/vladx/install.sh | bash

# Или загрузка и запуск установщика
wget https://your-website.com/vladx/install.sh
chmod +x install.sh
./install.sh
```

### Установка из конкретного ZIP архива

Вы можете указать URL для загрузки конкретного ZIP архива:

```bash
# Установка из конкретного ZIP архива
VLADX_DOWNLOAD_URL=https://your-website.com/vladx/vladx-custom.zip ./install.sh
```

### Автоматическая установка из репозитория (альтернативный метод)

Если вы хотите установить из репозитория:

```bash
curl -fsSL https://raw.githubusercontent.com/your-repo/vladx/main/install.sh | bash
```

Или скачайте и запустите установщик:

```bash
wget https://raw.githubusercontent.com/your-repo/vladx/main/install.sh
chmod +x install.sh
./install.sh
```

### Ручная установка

1. Убедитесь, что у вас установлен Node.js (версия 14 или выше)
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/your-repo/vladx.git
   cd vladx
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```
4. Установите глобально:
   ```bash
   npm install -g
   ```

## VladPM - Менеджер пакетов

VladPM - это менеджер пакетов для VladX, который позволяет устанавливать, удалять и управлять пакетами.

### Команды VladPM

```bash
vladpm install <пакет>     # Установить пакет
vladpm uninstall <пакет>   # Удалить пакет
vladpm list                # Показать установленные пакеты
vladpm search <пакет>      # Найти пакет
vladpm init                # Инициализировать проект
vladpm update              # Обновить пакеты
vladpm --version           # Показать версию
vladpm --help              # Показать справку
```

## Функция `запрос` в VladX

Функция `запрос` предоставляет возможность выполнять HTTP/HTTPS запросы к веб-сервисам и API.

### Синтаксис

```javascript
запрос(url, опции)
```

### Параметры

- `url` (обязательный) - URL-адрес для запроса
- `опции` (необязательный) - объект с параметрами запроса

### Опции запроса

- `method` - HTTP метод (GET, POST, PUT, DELETE и др.), по умолчанию "GET"
- `headers` - объект с HTTP заголовками
- `body` - тело запроса (для POST, PUT и др.)
- `raw` - если true, возвращает необработанный ответ

### Примеры использования

#### Простой GET запрос

```javascript
асинх функция получитьДанные() {
    пусть ответ = ждать запрос("https://api.example.com/data")
    вернуть ответ
}
```

#### POST запрос с телом

```javascript
асинх функция отправитьДанные(данные) {
    пусть ответ = ждать запрос("https://api.example.com/users", {
        method: "POST",
        headers: {
            "Content-Type": "application/json"
        },
        body: код(данные)  # код() - сериализует в JSON
    })
    вернуть ответ
}
```

#### Запрос с авторизацией

```javascript
асинх функция получитьСТокеном(токен) {
    пусть ответ = ждать запрос("https://api.example.com/protected", {
        headers: {
            "Authorization": "Bearer " + токен
        }
    })
    вернуть ответ
}
```

## Основы синтаксиса

VladX использует русский синтаксис для всех ключевых слов и встроенных функций:

```javascript
// Пример простой программы
пусть имя = "Влад"
печать("Привет, " + имя + "!")
```

### Комментарии

VladX поддерживает три типа комментариев:

```javascript
# Однострочный комментарий
// Также однострочный комментарий
/* 
   Многострочный 
   комментарий
*/
```

## Переменные

### Объявление переменных

VladX поддерживает два типа переменных:

```javascript
// Переменная (может быть изменена)
пусть имя = "Влад"
имя = "Анна"  # Это допустимо

// Константа (не может быть изменена)
конст возраст = 25
# возраст = 30  # Это вызовет ошибку
```

### Деструктуризация

VladX поддерживает деструктуризацию массивов и объектов:

```javascript
# Деструктуризация массива
пусть [первый, второй, третий] = [10, 20, 30]
печать(первый)  # Вывод: 10

# Деструктуризация объекта
пусть { имя, возраст } = { имя: "Влад", возраст: 30, город: "Москва" }
печать(имя)     # Вывод: Влад
печать(возраст) # Вывод: 30
```

## Типы данных

VladX поддерживает следующие типы данных:

- **Числа**: `5`, `3.14`, `-10`
- **Строки**: `"Привет"`, `'Мир'`
- **Логические значения**: `истина`, `ложь`
- **Массивы**: `[1, 2, 3]`, `["яблоко", "банан"]`
- **Объекты**: `{ имя: "Влад", возраст: 25 }`
- **Функции**: `функция() { ... }`
- **Пустое значение**: `ничто`

## Операторы

### Арифметические операторы

| Оператор | Описание | Пример |
|----------|----------|---------|
| `+` | Сложение | `5 + 3` |
| `-` | Вычитание | `5 - 3` |
| `*` | Умножение | `5 * 3` |
| `/` | Деление | `6 / 2` |
| `%` | Остаток от деления | `5 % 2` |
| `**` | Возведение в степень | `2 ** 3` |

### Операторы присваивания

| Оператор | Описание | Пример |
|----------|----------|---------|
| `=` | Простое присваивание | `x = 5` |
| `+=` | Сложение с присваиванием | `x += 3` |
| `-=` | Вычитание с присваиванием | `x -= 3` |
| `*=` | Умножение с присваиванием | `x *= 3` |
| `/=` | Деление с присваиванием | `x /= 3` |
| `%=` | Остаток с присваиванием | `x %= 3` |

### Битовые операторы

| Оператор | Описание | Пример |
|----------|----------|---------|
| `&` | Битовое И | `5 & 3` |
| `\|` | Битовое ИЛИ | `5 \| 3` |
| `^` | Битовое исключающее ИЛИ | `5 ^ 3` |
| `<<` | Сдвиг влево | `5 << 1` |
| `>>` | Сдвиг вправо | `5 >> 1` |

## Функции

### Объявление функций

```javascript
# Обычная функция
функция приветствовать(имя) {
    вернуть "Привет, " + имя + "!"
}

# Вызов функции
пусть результат = приветствовать("Влад")
печать(результат)  # Вывод: Привет, Влад!
```

### Функции с параметрами по умолчанию

```javascript
функция привет(имя, приветствие = "Привет") {
    вернуть приветствие + ", " + имя + "!"
}

печать(привет("Влад"))           # Вывод: Привет, Влад!
печать(привет("Анна", "Здравствуй")) # Вывод: Здравствуй, Анна!
```

### Функции с rest параметрами

```javascript
функция суммаВсех(...числа) {
    пусть сумма = 0
    для (пусть i = 0; i < длина(числа); i = i + 1) {
        сумма = сумма + числа[i]
    }
    вернуть сумма
}

печать(суммаВсех(1, 2, 3, 4, 5))  # Вывод: 15
```

### Асинхронные функции

```javascript
асинх функция получитьДанные() {
    # Имитация асинхронной операции
    ждать(1000)  # Ждем 1 секунду
    вернуть "Данные получены"
}

асинх функция основная() {
    пусть данные = ждать получитьДанные()
    печать(данные)
}
```

## Управляющие структуры

### Условные выражения

```javascript
# Простое условие
если (возраст >= 18) {
    печать("Совершеннолетний")
} иначе {
    печать("Несовершеннолетний")
}

# Множественные условия
если (оценка == 5) {
    печать("Отлично!")
} иначеесли (оценка == 4) {
    печать("Хорошо")
} иначеесли (оценка == 3) {
    печать("Удовлетворительно")
} иначе {
    печать("Неудовлетворительно")
}

# Тернарный оператор
пусть результат = (возраст >= 18) ? "совершеннолетний" : "несовершеннолетний"
```

### Циклы

#### Цикл while

```javascript
пусть i = 0
пока (i < 5) {
    печать("Итерация: " + i)
    i = i + 1
}
```

#### Цикл for

```javascript
# Классический for
для (пусть i = 0; i < 5; i = i + 1) {
    печать("Число: " + i)
}

# Цикл по массиву
пусть фрукты = ["яблоко", "банан", "апельсин"]
для (пусть i = 0; i < длина(фрукты); i = i + 1) {
    печать(фрукты[i])
}
```

### Switch-выражения

```javascript
выбор (оценка) {
    когда 5: {
        печать("Отлично!")
    }
    когда 4: {
        печать("Хорошо")
    }
    когда 3: {
        печать("Удовлетворительно")
    }
    поумолчанию: {
        печать("Нужно постараться")
    }
}
```

## Встроенные функции

### Функции для работы с консолью

- `печать(...аргументы)` - выводит значения в консоль
- `вывод(...аргументы)` - выводит значения без перевода строки
- `ошибка(...аргументы)` - выводит ошибку в консоль

### Математические функции

- `максимум(...числа)` - возвращает максимальное число
- `минимум(...числа)` - возвращает минимальное число
- `случайный()` - возвращает случайное число от 0 до 1
- `случайноеЦелое(мин, макс)` - возвращает случайное целое число в диапазоне
- `округлить(число)` - округляет число
- `корень(число)` - квадратный корень
- `степень(основание, показатель)` - возведение в степень

### Функции для работы со строками

- `длина(строка)` - длина строки
- `нижнийРегистр(строка)` - преобразование в нижний регистр
- `верхнийРегистр(строка)` - преобразование в верхний регистр
- `обрезка(строка)` - удаление пробелов по краям
- `заменить(строка, старый, новый)` - замена подстроки
- `разделить(строка, разделитель)` - разделение строки на массив
- `соединить(массив, разделитель)` - соединение массива в строку

### Функции для работы с массивами

- `размер(массив)` - количество элементов в массиве
- `первый(массив)` - первый элемент
- `последний(массив)` - последний элемент
- `добавить(массив, элемент)` - добавление элемента
- `удалить(массив, индекс)` - удаление элемента по индексу
- `фильтр(массив, функция)` - фильтрация элементов
- `отобразить(массив, функция)` - преобразование элементов
- `сортировать(массив, компаратор)` - сортировка

### Функции для работы с объектами

- `ключи(объект)` - массив ключей объекта
- `значения(объект)` - массив значений объекта
- `пары(объект)` - массив пар [ключ, значение]
- `слить(...объекты)` - объединение объектов

### Функции для работы с файлами

- `файлЧитать(путь)` - чтение содержимого файла
- `файлЗаписать(путь, содержимое)` - запись в файл
- `файлСуществует(путь)` - проверка существования файла

### Функции для работы с датой и временем

- `время()` - текущее время в миллисекундах
- `дата()` - текущая дата
- `форматДаты(дата, формат)` - форматирование даты

### Функции для работы с JSON

- `код(объект)` - сериализация в JSON
- `разкод(строка)` - десериализация из JSON

### Функции для работы с сетью

- `запрос(url, опции)` - HTTP запрос

### Функции для работы с криптографией

- `хешMD5(строка)` - MD5 хеш
- `хешSHA256(строка)` - SHA-256 хеш
- `генUUID()` - генерация UUID

## Примеры

### Простой калькулятор

```javascript
функция калькулятор(операция, a, b) {
    выбор (операция) {
        когда "сложить": {
            вернуть a + b
        }
        когда "вычесть": {
            вернуть a - b
        }
        когда "умножить": {
            вернуть a * b
        }
        когда "разделить": {
            если (b != 0) {
                вернуть a / b
            } иначе {
                вернуть "Ошибка: деление на ноль"
            }
        }
        поумолчанию: {
            вернуть "Неизвестная операция"
        }
    }
}

печать(калькулятор("сложить", 5, 3))    # Вывод: 8
печать(калькулятор("разделить", 10, 2)) # Вывод: 5
```

### Асинхронная обработка данных

```javascript
асинх функция обработатьДанные() {
    # Имитация загрузки данных
    пусть данные = ждать запрос("https://api.example.com/users")
    
    # Обработка данных
    пусть активныеПользователи = фильтр(данные, функция(пользователь) {
        вернуть пользователь.активный
    })
    
    вернуть активныеПользователи
}

асинх функция основная() {
    попытка {
        пусть результат = ждать обработатьДанные()
        печать("Найдено " + размер(результат) + " активных пользователей")
    } перехват (ошибка) {
        печать("Ошибка при обработке данных: " + ошибка)
    }
}

# Вызов основной функции
основная()
```
